Questions

Networking FAQs

Do you supply the hardware?

We can, or we can work with hardware you already own. Hardware is always billed at cost with no markup.

Can you set up a small office network?

Yes — small office network setup starts at $149 and includes multi-device configuration.

Do you install Starlink?

Yes — mounting the dish, running cable into the house, and integrating it with your existing router or mesh network so the rest of your home network works exactly as it did before.

What is MoCA networking, and when do I need it?

MoCA uses your home's existing coax (TV) wiring to carry a wired network connection between rooms — a good option when running new ethernet isn't practical but Wi-Fi alone isn't reliable enough, e.g. for a streaming box or a mesh access point in a hard-to-reach room.
